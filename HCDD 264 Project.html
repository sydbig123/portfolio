Moving Forward

If my group and I were to move forward with this project, we would make the following edits to our project before continuing with user-testing.

  • Ensure the entire application uses the elements, styles, and fonts listed in the Style Guide
  • Make button areas larger and ensure all buttons are in places where they will not be overlooked by users
  • Create more visual feedback within the application for successful and unsuccessful actions
